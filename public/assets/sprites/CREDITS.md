# Sprite credits

The character sprites are from the **"DungeonTileset II"** by **0x72**,
released under the **Creative Commons Zero (CC0 / public domain)** license.

- Source: https://0x72.itch.io/dungeontileset-ii
- Atlas files (`fauna`, `lizard`) obtained via the CC0 starter project
  https://github.com/ourcade/phaser3-dungeon-crawler-starter

CC0 means no attribution is legally required — this note is provided as courtesy.

All other in-game textures (projectiles, XP gems, coins, effects) are generated
procedurally at runtime in `src/scenes/PreloadScene.ts`.
